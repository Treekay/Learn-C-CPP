# [2016 C Midterm B] Decimal To Hexadecimal

# Decimal To Hexadecimal

## Problem

Given a decimal number, output it in hexadecimal in **inverse order**.

## Detail

Use **upper-case letter** for the number larger than 9.

### Sample Input 1

```
16
```

### Sample Output 1

```
01

```

### Sample Input 2

```
26
```

### Sample Output 2

```
A1

```

### Sample Input 3
```
288
```

### Sample Output 3

```
021

```
